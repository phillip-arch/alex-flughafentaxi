# Alex Flughafentaxi

Next.js application for booking airport taxi rides in Vienna.

## Development

npm install
npm run dev

## Deployment

Deployed via Vercel. Environment variables are configured in Vercel Project Settings.

Required public browser key for address search:

```text
NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=...
```

Enable Google Maps JavaScript API and Places API for this key.
